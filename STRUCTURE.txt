FICHIERS CLÉS:

Backend:
- server.js                → Serveur Express principal
- cron/priceChecker.js    → Vérifie les prix 2x/jour
- services/flightAPI.js   → Intégration Duffel/FlightLabs
- services/twilioService.js → Envoi SMS
- models/database.js      → Gestion SQLite

Frontend:
- App.js                  → Application principale
- components/Dashboard.js → Page principale
- components/DestinationCard.js → Carte de destination
- components/PriceChart.js → Graphique historique
- services/api.js         → Client API

Database:
- init.js                 → Crée tables et données test
- tracker.db              → Base SQLite avec destinations

Configuration:
- backend/.env            → Clés API (Duffel, Twilio, SMTP)
- frontend/package.json   → Dépendances React
- backend/package.json    → Dépendances Node.js
